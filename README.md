# Relink-happ-flow

HTML-страница для автоматического перенаправления в приложение **FlowVPN** через кастомный протокол `happ://`.

## Описание

Этот проект предоставляет веб-страницу, которая:
- Извлекает параметр `key` из URL
- Автоматически открывает приложение FlowVPN с указанной конфигурацией
- Предоставляет кнопку-фолбэк для ручного открытия приложения

## Использование

Откройте `index.html` в браузере с параметром `key`:

```
index.html?key=YOUR_VPN_KEY
```

Или разместите файл на веб-сервере и используйте URL вида:

```
https://your-domain.com/index.html?key=YOUR_VPN_KEY
```

## Как это работает

1. Скрипт извлекает параметр `key` из query-строки URL
2. Формирует ссылку вида `happ://add/{key}`
3. Пытается открыть приложение FlowVPN автоматически
4. Если автоматическое открытие не сработало — показывает кнопку для ручного запуска

## Особенности

- **Адаптивный дизайн** — корректно отображается на мобильных устройствах, планшетах и десктопах
- **Два метода открытия** — через создание ссылки и через `location.href` для максимальной совместимости
- **Fallback-кнопка** — если автоматическое открытие не сработало

- Приложение **FlowVPN** должно быть установлено на устройстве
- Устройство должно поддерживать кастомные протоколы (`happ://`)
